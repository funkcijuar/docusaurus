# Docusaurus v2

## Installation

Install dependencies for v2 dummy website and v2 package itself
```bash
cd ../v2-website && yarn
cd v2 && yarn
```

## Development Server

```bash
yarn start # open http://localhost:3000/ 
```

## Production Build

```bash
cd ../v2-website && yarn build # check ../v2-website/build
```

## Test

```bash
yarn test
```
